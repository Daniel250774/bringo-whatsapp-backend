Notă: această versiune trimite prin WhatsApp Cloud API și sincronizează cardurile cu backend-ul Render. Livratorul poate scrie Gift către numărul API și primește automat primul card disponibil. Administratorul primește copie cu data/ora și numărul de gifturi rămase. După primire, livratorul poate intra automat într-o pauză temporară configurabilă, ca să nu poată cere două gifturi prea repede. Pe telefon folosește „Curăță telefonul”, apoi „Încarcă din backend” ca să aduci aceeași evidență de pe calculator.
